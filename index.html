AI For Problem Solving

Languages used: Java

This is a 3rd year project in which I was tasked with implementing an A* search algorithm to solve a puzzle using a given aips library. The puzzle contained 3 cities (A,B,C) where there are fixed costs of travel between each city and a cost of moving cargo per ton per city. The truck has a maximum load. We had to intake an initial state and a goal state. The search was implemented using different classes and functions some key ones were: - Successor function: Gathered all possible next moves - ApplyAction function: was used so the actions could be applied to the current state - Heuristic function: This had to never over-estimate the cost otherwise the A* search would not work Overall Module Grade: TBC

Airport Device

Languages used: Python

This is a 2nd year Internet Of Things project and was created in a team of myself and 2 others. This consists of 3 different devices that all make use of a raspberry pi and connected devices. These were all linked together and communicated data between each of them. The Baggage Device subscribes to MQTT (sonica/baggage) to recieve messages published and then further publishes and forwards data to Adafruit in which it is stored. This device is used to scan baggage tags and direct baggage to the appropriate gate via conveyor. The Weather Device pushes to MQTT (sonica/water) to send messages about water detection and other weather data. All data collected is then pushed to the Adafruit dashboard which allows for the end user to have an overall view of the airport sensors The Air Traffic Device, detects aircraft and guides them to where they need to be or go. This data is then sent to Adafruit to create a dashboard with this information. Overall Module Grade: C
