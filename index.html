Airport Device

Languages used: Python

  • Internet of Things (IoT) project using Raspberry Pi
  • Team project (3 members)
  • Devices involved:
    • Baggage Device
      • Subscribes to MQTT (sonica/baggage) for baggage tag scanning
      • Publishes data to Adafruit for storage and visualization
      • Used to direct baggage to the correct gate via conveyor
    • Weather Device
      • Pushes data to MQTT (sonica/weather) about weather conditions
      • Collects water detection and environmental sensor data
      • Publishes data to Adafruit dashboard for airport monitoring
    • Air Traffic Device
      • Detects aircraft movement and assigns appropriate routes
      • Communicates findings to Adafruit for dashboard visualization
  • Overall Module Grade: C
